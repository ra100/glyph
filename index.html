Hello agents!

In this bundle you may find usefull tool for Glyph sequence analysis.

The Search Glyph tool is most major part of bundle. This tool alows you to draw a part of glyph and either find the known glyphs that contains same strokes, or find, what glyphs are hidden (combined) inside your one.

There are two modes of operation:

Search
Draw the part of glyph. The glyphs that contains your strokes will be revealed. The empty drawing matches all glyphs
Match
Draw the new glyph that you have found. It will show you, what combination of known glyphs makes your drawing. glyphs. E.g. if you will draw Open All, it will show you Open and All glyphs

Unfortunately, I didn't put all the glyphs to current database. Also, the code I've written is buggy. But feel free to clone the project on GitHub, and provide patches.

For those, who want to add glyphs to database, there is a Drawing tool that allows to add new items and copy it to sequence file.
